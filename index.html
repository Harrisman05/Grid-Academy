grid-template-columns

grid-template-columns: 2fr 1fr;

Creates grid template with 2 columns --- first column takes up 66% space, 2nd column takes up 33% space

grid-template-columns: repeat(3, 300px);

Creates grid template with 3 columns, each with 300px width

grid-auto-rows: 150px;

No matter how many rows there are, they will always be 100px tall

grid-auto-rows: minmax(150px, auto);

minmax allows for setting a minimum row height, and preventing text overspilling container by having max as auto
